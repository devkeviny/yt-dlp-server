Technical Reference

API Documentation

Interface de integração para automação de downloads e monitoramento de node.

vpn_key Autenticação

Todas as requisições devem incluir a chave de API no cabeçalho HTTP.

X-API-Key: your_secret_api_key_here
GET

/info

Analisa URL

Retorna metadados do vídeo (Título, Duração, Thumbnail) e valida a disponibilidade via proxy.

GET /info?url=https://youtube.com/watch?v=...&net=auto
GET

/download

Processa Arquivo

Inicia o stream de download do arquivo no formato especificado.

GET /download?url=...&fmt=mp4&net=auto
GET

/api/system/stats

Saúde do Node

Retorna a utilização de CPU, RAM, Disco e Uptime do servidor em tempo real.

GET /api/system/stats
